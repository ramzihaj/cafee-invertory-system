## 🆘 Dépannage

### Erreur "Cannot find module"
```bash
cd server
npm install
```

### Base de données corrompue
```bash
rm database/*.sqlite
node seeders/seed-full-products.js
```

### Port déjà utilisé
Changez le port dans `.env`
